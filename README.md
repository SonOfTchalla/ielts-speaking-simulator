# IELTS Speaking Simulator

A real-time IELTS Speaking Test simulator with AI-powered feedback.

## Features
- Real-time conversation with an AI examiner.
- IELTS scoring based on fluency, pronunciation, grammar, and vocabulary.
- Practice Mode and Test Mode.
- Detailed feedback and downloadable PDF reports.

## Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/ielts-speaking-simulator.git
   cd ielts-speaking-simulator

2. Install dependencies:
    ```bash
    pip install -r requirements.txt

3. Run the app:
    ```bash
    streamlit run app/main.py
